Swift GRB Alert Catcher

Эта программа каждые 15 сек. проверяет RSS Swift'a. Если там появляется новый гамма-всплеск, который может наблюдаться в данный момент на телескопе, открывается браузер с карточкой и координатами всплеска.

Конфигурационный файл D:\dmb\GRBAlert\config.cfg

Чтобы убрать программу из автозагрузки удалите ярлык GA в Start => All Programs => Startup

Иконка в трее Значение
новых гамма-всплесков нет, RSS доступна, идёт сканирование
RSS недоступна, возможно проблемы с интернетом или сайтом skyalert.org
появился новый гамма-всплеск, который, вероятно, доступен для наблюдений,
нажмите reset в меню, чтобы сбросить это состояние

При появлении нового доступного всплеска в браузере открывается 3 страницы:
- ссылка на событие на SkyAlert, например
- BAT position. Это не точные координаты с телескопа BAT. Однако уже можно начинать наводиться в эту область.
- UVOT position. Через некоторое время появляются точные координаты с UVOT. Вот эту область с конечными координатами и надо снимать.

Пример того, как выглядит страница с поисковой картой можно посмотреть, нажав кнопку (пункт меню) "Show last"
